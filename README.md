## Project Settings

### Install Packages
`yarn`

### Linking React-Native libraries and assets
`npx react-native link`

### Start React-Native
`yarn start`

### Run Android
`yarn android`

### Setting for iOS
`cd ./ios && pod install && cd ../`

### Run iOS
`yarn ios`
